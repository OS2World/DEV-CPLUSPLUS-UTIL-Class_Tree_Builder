     Программа Class Tree Builder предназначена для визуализации
  иерархии наследования классов в программах на C++. Программа
  является FreeWare и распространяется "as is". Как и следует ожидать
  никаких гарантий.

  Вероятно (если не будет найдено новых ошибок) эта версия станет последней.
  DOS версия не сопровождается, не поддерживается, не расширяется, даже
  не фиксится (у меня просто нет досовского компилятора и, я надеюсь, больше
  никогда не будет). Все что будет появлятся - будет появлятся под OS/2 и,
  не исключено, под Linux. Windows приравнивается к DOS ;)

                        Вступление.

     Тем кто пишет на C++ хорошо известно, что по мере того как иерерхия
  классов растет, становиться трудно удерживать ее в голове, особенно
  если при этом используется множественное наследование. Да и вообще
  хочется иногда остановиться и окинуть взглядом все что уже сделано.
     Аналогичная проблема возникает при попытке разобраться в чужой
  библиотеке классов. Хорошо если есть какая-нибудь документация,
  в которой приведена иерархия, но так бывает не всегда :(. Кроме
  того : если такая документация есть и в ней приведена иерархия,
  зачастую необходимо знать где находится декларация класса.
     Для этих целей и был написан Class Tree Builder.

                Описание ключей программы.

     Командная строка программы:
  CTB [-{iwsnma} [-Ipath{;path}] [-Sname] [-oname[.def]] [path\]filespec[.h]
  
  -i     Указавает CTB игнорировать все директивы #include
         Это заметно увеличивает скорость в тех случаях когда все описания
         находятся в одном файле, который кроме того содержит еще директивы
         #include для системных файлов (например windows.h ;).

  -w     Выключает печать предупреждений.

  -s     Включает печать данных о месте определения класса.
         Печатается строка и имя файла в котором приводится
         полное определение класса.

  -Ipath Указывает пути поиска файлов. Возможно указание нескольких
         путей через точку с запятой. Помимо этого используется информация из
         переменной окружения INCLUDE, если она определена.

  -Sname Указывает CTB, что при выводе иерархии необходимо начинать вывод
         с класса с именем name.

  -oname Вывод будет произведен в файл с именем name.
         По умолчанию вывод делается на экран.
         Усли не указано расширение то по умолчанию файл будет
         иметь расширение .def.

  -c     Понимать объявления 'struct' и 'union' как 'class'

  -a     Печатать все обнаруженые объявления.
         Если режим не задан, то будут печататься только классы
         имеющие наследников.

  -n     Не рисовать дерево. Удобна соместно с -m -a.
  -m     Печать методов класса.
